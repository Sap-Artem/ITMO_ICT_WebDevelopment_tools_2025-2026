# Personal Finance Service

## Описание проекта

Personal Finance Service — серверное приложение на FastAPI для управления личными финансами.

Сервис позволяет пользователю:

- регистрироваться и авторизовываться;
- создавать доходы и расходы;
- распределять операции по категориям;
- добавлять теги к транзакциям;
- устанавливать бюджеты;
- создавать финансовые цели;
- просматривать финансовую сводку;
- получать информацию о превышении бюджета.

---

## Используемые технологии

- FastAPI
- SQLModel
- PostgreSQL
- Alembic
- JWT
- Passlib
- Python-dotenv
- Pydantic / SQLModel schemas

---

## Основной функционал

В проекте реализованы:

- регистрация пользователя;
- авторизация пользователя;
- генерация JWT-токена;
- аутентификация по JWT;
- хэширование паролей;
- смена пароля;
- CRUD для категорий;
- CRUD для тегов;
- CRUD для транзакций;
- CRUD для бюджетов;
- CRUD для финансовых целей;
- many-to-many связь между транзакциями и тегами;
- отчет по доходам, расходам, балансу и превышенным бюджетам.
