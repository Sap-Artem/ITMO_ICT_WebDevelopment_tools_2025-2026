# Personal Finance Service

Сервис для управления личными финансами на FastAPI.

## Возможности

- Добавление доходов и расходов
- Категории транзакций
- Работа с бюджетами
- Анализ финансов
- CRUD API
- PostgreSQL
- JWT-аутентификация

## Технологии

- FastAPI
- SQLModel
- PostgreSQL
- Alembic
- JWT
